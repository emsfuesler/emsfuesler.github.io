AstroNav

GSpeak Arduino interactive installation made in collaboration width Oblong Industries. This project uses a captian/navigator dynamic to explore celestial bodies. The navigator hovers their hand over the astrological sign, and if the captian hits the approval button, the GSPeak code allows you to travel through the celestial body. Another Arduino sensor can also be used to navigate around the celestial body and view it from different angles. This project was made with Anissa Santos, Remi Wedin, and Eumin Lee.
