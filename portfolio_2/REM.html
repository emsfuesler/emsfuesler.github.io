rapidEyeMovement

rapidEyeMovement was an installation project attempting to create experiences of synesthesia. This installation was set up in a room with mirror vinyls and led changing clouds. On the wall was a code generated projection which followed the viewer around the room using a camera and frame differencing. The code uses sound analysis to create visuals for an original dreamscape made in Adobe Audition.
